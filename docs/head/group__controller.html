bool rdmctl_is_non_disc_response ( const RdmBuffer *  buffer)

Determine whether a packed RDM message is a non-discovery RDM command response.

More specifically, whether the command class of the response is one of GET_COMMAND_RESPONSE or SET_COMMAND_RESPONSE.

Parameters
[in]bufferThe packed RDM message.
Returns
true (the message is a valid non-discovery RDM command response) or false (the message is invalid RDM or not a non-discovery command response).
